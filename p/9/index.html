回顾

本文内容同样适用于“省电”这一话题，但没有心思再去修改原文，且这个模块也值得我专门来写一篇文章来介绍.

如何解决耗电崩的问题

如果你正在使用XP框架 但认为XP框架会占用系统资源，增加耗电，甚至说拖慢系统性能，你可以参考这篇文章

Xposed会影响性能及耗电？

将QQ微信加入缓存

加入乖巧模式后台几乎0耗电的

请先确认你的手机里已经安装了Xposed框架及应用控制器+应用管理XP模块/捐赠版

打开基础-关联启动限制并开启后在右上角找到关联启动规则后添加以下规则

//微信规则
ALLOW android com.tencent.mm
ALLOW com.tencent.mm com.tencent.mm
DENY * com.tencent.mm

//QQ规则
ALLOW android com.tencent.mobileqq
ALLOW com.tencent.mobileqq com.tencent.mobileqq
DENY * com.tencent.mobileqq

//tim规则
ALLOW android com.tencent.tim
ALLOW com.tencent.tim com.tencent.tim
DENY * com.tencent.tim

再回到上一页面 找到右下角的+号 选择微信/TIM/QQ 加入后选中 配置单项配置

然后回到主页-基础-自启限制 打开 主页-进阶-乖巧模式-启用

到这里还没有结束 接下来打开应用控制器 在侧边栏中找到-应用锁、启动控制、常驻内存后找到微信/TIM/QQ的内存常驻打开#如果你使用的是应用管理XP捐赠版可以直接在基础中找到后台保护功能

之后就可以体验划卡秒进占用内存减少 但消息却不会延迟的操作啦

待机耗电

在状态中找到内存清理后打开 添加所有应用 在右上角的设置中选择”休眠代替强行停止“即可

然后再找到-进阶-打盹 启用 在下面配置中找到强行进入并打开即可

再配合应用控制器的所有应用全部禁唤醒锁【通知应用除外】效果显著

GCM代收

添加以上规则后请确定已安装谷歌服务并已经登录 已经安装谷歌版微信

在进阶中开启GCM代收即可 但经过上面的配置我个人认为没什么特别大的作用了
