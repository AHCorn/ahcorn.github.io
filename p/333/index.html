Typecho主题案例—Admin主题

Mar 30, 2020
阅读时长: 3 分钟

看起来很难，但实际上手后你才会发现，这一切都很简单。

扒后台

直接 Ctrl+S 把Typecho的后台扒下来

主题化

然后在你的 Themes 文件夹内新建一个Admin文件夹，然后新建一个 index.php 把这段粘贴到开头

 1<?php
 2/**
 3 * 基于Typecho后台写的一款主题。
 4 * 
 5 * @package Admin
 6 * @author 安和
 7 * @version 1.0
 8 * @link http://lolicorn.com
 9 */
10if (!defined('__TYPECHO_ROOT_DIR__')) exit;?>

然后出去新建个 head.php 粘贴刚刚复制到的头部代码，再把原来的这个路径改成调用主题地址，公共库有的就不要调用本地的，最后把标题改成调用当前页面

 1<html class="no-js"><head>
 2<meta http-equiv="content-type" content="text/html; charset=UTF-8">
 3<meta charset="UTF-8">
 4<meta http-equiv="X-UA-Compatible" content="IE=edge">
 5<meta name="renderer" content="webkit">
 6<meta name="viewport" content="width=device-width, initial-scale=1">
 7<title><?php $this->archiveTitle(array( 'category' => _t('分类 %s 下的文章'), 'search' => _t('包含关键字 %s 的文章'), 'tag' => _t('标签 %s 下的文章'), 'author' => _t('%s 发布的文章') ), '', ' - '); ?><?php $this->options->title(); ?><?php if($this->is('index')): ?> - <?php $this->options->description() ?><?php endif; ?></title><!--输出当前标题 -->
 8<link rel="stylesheet" href="https://cdn.staticfile.org/normalize/8.0.1/normalize.css">
 9<link rel="stylesheet" href="<?php $this->options->themeUrl('css/style.css'); ?>">
10<link rel="stylesheet" href="<?php $this->options->themeUrl('css/grid.css'); ?>">
11<!--把调用的样式改成主题文件夹 -->
12<style>@media print {#ghostery-purple-box {display:none !important}}</style></head>

然后新建一个 header.php 把那个黑条条的代码弄进来 然后一样是需要做一些修改 删了控制台，然后把概要改成返回首页

 1<!--删掉多余的内容 -->
 2<div class="typecho-head-nav clearfix" role="navigation">
 3<nav id="typecho-nav-list">
 4<ul class="root">
 5<li class="parent"><a href="<?php $this->options->siteUrl(); ?>">概要</a></li>
 6<!-- 概要改成返回首页 也可以直接改成首页这些hhh -->
 7</ul>
 8<ul class="root">
 9<li class="parent"><a href="<?php $this->options->siteUrl(); ?>/post.html">文章</a></li>
10<!-- 这里后续做一个独立页面 用来显示全部文章，一样的道理，直接扒后台的就行 -->
11</ul>
12<?php $this->widget('Widget_Contents_Page_List')->to($pages); ?>
13<?php while($pages->next()): ?>
14<ul class="root">
15<li class="parent"><a href="<?php $pages->permalink(); ?>" title="<?php $pages->title(); ?>"><?php $pages->title(); ?></a></li>
16</ul>
17<?php endwhile; ?> <!-- 输出自定义页面 -->
18<!-- 把下面的用户删掉 有需要可以加点别的 或者弄个登录用户判断 我看到feed大佬那里是搞了个游客 -->

这下顶栏OK之后直接搞首页内容就行了

加上两行调用

1$this->need('heade.php');
2$this->need('header.php');

然后粘贴 并修改下首页

 1<div class="main">
 2<div class="container typecho-dashboard">
 3<div class="typecho-page-title">
 4<h2>网站概要</h2>
 5</div>
 6<div class="col-mb-12 welcome-board" role="main">
 7              <?php Typecho_Widget::widget('Widget_Stat')->to($stat); ?>
 8               <p>目前有 <em><?php $stat->publishedPostsNum() ?></em> 篇文章, 并有 <em><?php $stat->publishedCommentsNum() ?></em> 条评论在 <em><?php $stat->categoriesNum() ?></em> 个分类中. </p>
 9</div>

15                    <ul>
16                    <?php if($posts->have()): ?>
17                    <?php while($posts->next()): ?>
18                        <li>
19                            <span><?php $posts->date('n.j'); ?></span>
20                            <a href="<?php $posts->permalink(); ?>" class="title"><?php $posts->title(); ?></a>
21                        </li>

50<h3>友情链接</h3>

页脚

把底下的版权信息和js调用弄到页脚 新建一个footer.php

 1<!-- 页脚 -->
 2<!-- 版权声明 -->
 3<div class="typecho-foot" role="contentinfo">
 4<div class="copyright">
 5<a href="http://typecho.org/" class="i-logo-s" target="_blank">Typecho</a>
 6<p>由 <a href="http://typecho.org/" target="_blank">Typecho</a> 强力驱动</p>
 7<p>Theme By <a href="https://lolicorn.com/" target="_blank">安和</a></p>
 8<!-- 改成自己的就行啦 反正我也没费多少功夫哈哈哈 -->
 9</div>
10</div>
11<!-- 引用js文件 这里是在保存的后台页面的底下 -->
12<script src="https://cdn.staticfile.org/jquery/3.4.1/jquery.js"></script>
13<script src="https://cdn.staticfile.org/jqueryui/1.12.1/jquery-ui.js"></script>
14<script src="<?php $this->options->themeUrl('/js/typecho.css');?>"></script>

然后在index.php里调用

1<?php $this->need('header.php'); ?>

这样 首页就OK了 （目前咕咕咕 有空再写
